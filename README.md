# Analiza finansowa Sygnity S.A. (2022–2024) / Financial Analysis of Sygnity S.A.

Projekt zaliczeniowy wykonany w MS Excel. Kompleksowa analiza finansowa polskiej spółki IT notowanej na Giełdzie Papierów Wartościowych w Warszawie.

## 🇵🇱 Opis

Analiza finansowa Sygnity S.A. za lata 2022–2024 obejmuje badanie kondycji finansowej spółki na podstawie publicznie dostępnych sprawozdań finansowych.

### Zakres analizy
- Analiza bilansu – struktura aktywów i pasywów
- Analiza rachunku zysków i strat
- Analiza wskaźnikowa (płynność, rentowność, zadłużenie, sprawność)
- Obliczanie struktury i dynamiki zmian rok do roku

### Technologie
`MS Excel` `tabele przestawne` `wykresy` `formuły finansowe`

---

## 🇬🇧 Description

A university project developed in MS Excel. Comprehensive financial analysis of Sygnity S.A., a Polish IT company listed on the Warsaw Stock Exchange, covering fiscal years 2022–2024.

### Scope
- Balance sheet analysis – asset and liability structure
- Income statement analysis
- Financial ratio analysis (liquidity, profitability, debt, efficiency)
- Year-on-year structure and dynamics calculations

### Technologies
`MS Excel` `pivot tables` `charts` `financial formulas`
